About macUSB

macUSB is a one-click tool that turns your modern Mac (Apple Silicon or Intel) into a service machine capable of reviving older Apple computers — without Terminal commands, disk-format guessing, or wrestling with legacy installer quirks.

Why it exists

With Apple’s transition to Apple Silicon, preparing installation media for older Intel- and PowerPC-based Macs became increasingly painful. Users often run into Terminal errors, expired certificates in older installers, or toolchain incompatibilities. macUSB automates the full USB creation flow so you can focus on installing the system, not debugging the process.

What you get

Legacy support

Create installers for systems over a decade old directly on a modern Mac.

Time saving

macUSB detects the system version in your installer file, formats the drive, and copies everything automatically.

Auto fixes

For selected installers (e.g., High Sierra / Mojave), macUSB applies fixes so they keep working despite expired Apple certificates.

Security

The application is officially notarized by Apple. This guarantees that the software is free from malicious code and ensures a seamless, secure first launch on macOS without requiring any extra security workarounds.

PowerPC revival

macUSB can format USB drives using Apple Partition Map (APM) and transfer installers for PowerPC Macs — no CD/DVD or FireWire required.

Requirements

Host Mac

  • Processor: Apple Silicon or Intel
  • System: macOS Sonoma 14.6 or newer
  • Free storage: at least 15 GB to build installers

USB drives & external storage

  • Capacity: minimum 16 GB
  • Speed: USB 3.0+ recommended
  • External drives (HDD/SSD): enable Options → Enable external drives support

Supported installer file formats

macUSB supports disk images (.dmg, .cdr, .iso) and raw installer apps (.app).
