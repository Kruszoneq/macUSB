About macUSB

macUSB is built to make bootable macOS and Mac OS X media creation practical again on modern hardware, without scripting and without guesswork.

Project intent

The project exists to bridge an increasingly difficult gap: many technicians and enthusiasts now work from Apple Silicon Macs, while the machines they need to recover often rely on legacy installer formats and older boot paths. macUSB turns that complexity into one guided workflow.

Why this matters now

Preparing legacy installers manually on modern macOS can fail for many reasons: incompatible workflows, expired certificate-era edge cases, formatting mismatches, and brittle one-off terminal commands. macUSB removes those blockers so you can focus on the target machine, not low-level setup steps.

What v2.0 brings

Native privileged helper

Creation is handled through a native helper architecture, improving reliability and reducing dependence on external terminal sessions.

Automatic media preparation

macUSB verifies USB partition schemes and formats up front, then prepares media for a stable installer write path.

Interactive progress dashboard

Track pending, active, and completed stages in real time with clearer status visibility during creation.

Operational clarity

Write speed indicators, completion notifications, and richer logs make the process easier to trust and troubleshoot.

Supported recovery scenarios

Current macOS installers

Create USB installers for current macOS releases from a modern host Mac with a predictable workflow.

Legacy Intel systems

Prepare media for older Intel Macs where installer preparation is often inconsistent on current systems.

PowerPC revival

macUSB supports Tiger and Leopard workflows for PowerPC-focused restoration, with direct links to Open Firmware USB boot guidance.

PowerPC USB Boot Guide

Open-source and safety

macUSB is open-source and released under the MIT License. Distribution and update messaging prioritize transparency, and the application is Apple-notarized for safer first-run experience on macOS.
